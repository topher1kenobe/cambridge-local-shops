== Description ==
A child theme for GeneratePress that includes template files for the Simple Small Business plugin that holds information for businesses dealing with the Coronavirus.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. At the top of the page click the Upload Theme button
3. Upload this theme. WordPress will automatically install GeneratePress at the same time.
4. Click on the 'Activate' button to use your new theme right away.
